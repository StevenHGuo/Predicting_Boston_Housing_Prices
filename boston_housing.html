编程练习3：定义衡量标准¶

如果不能对模型的训练和测试的表现进行量化地评估，我们就很难衡量模型的好坏。通常我们会定义一些衡量标准，这些标准可以通过对某些误差或者拟合程度的计算来得到。在这个项目中，你将通过运算决定系数 R2 来量化模型的表现。模型的决定系数是回归分析中十分常用的统计信息，经常被当作衡量模型预测能力好坏的标准。

R2的数值范围从0至1，表示目标变量的预测值和实际值之间的相关程度平方的百分比。一个模型的R2 值为0还不如直接用平均值来预测效果好；而一个R2 值为1的模型则可以对目标变量进行完美的预测。从0至1之间的数值，则表示该模型中目标变量中有百分之多少能够用特征来解释。模型也可能出现负值的R2，这种情况下模型所做预测有时会比直接计算目标变量的平均值差很多。

在下方代码的 performance_metric 函数中，你要实现：

  • 使用 sklearn.metrics 中的 r2_score 来计算 y_true 和 y_predict的R2值，作为对其表现的评判。
  • 将他们的表现评分储存到score变量中。

或

  • (可选) 不使用任何外部库，参考决定系数的定义进行计算，这也可以帮助你更好的理解决定系数在什么情况下等于0或等于1。

def performance_metric2(y_true, y_predict):
    """计算并返回预测值相比于预测值的分数"""
    
    score = None
    y_true_mean = np.mean(y_true)
    
    SST = sum( (y_i - y_true_mean)**2 for y_i in y_true)
    SSR = sum( (y_i - f_i)**2 for y_i, f_i in zip(y_true, y_predict))
    
    score = round(1 - SSR/SST, 3)
    return score

问题 7- 网格搜索（Grid Search）¶

什么是网格搜索法？如何用它来优化模型？

问题 7 - 回答:¶

  • 网格搜索法是将数据按照特征值进行分类，获取最佳的特征值。通过网格搜索获取的分类特征值基本是全局最优解，可避免重大误差。
  • 网格搜索的是超参数。获取用户给定的框架参数（非该框架可以设置的所有参数）的所有组合，然后通过特定的评价函数（例如决定系数）进行比对，获取在该评价函数下最优参数集合即为最终我们需要的模型参数。

  • GridSearchCV中的'cv_results_'属性能告诉我们什么？

问题 8 - 回答：¶

  • K折交叉验证法是将全部数据分成k-1个训练集，1个测试集，总共k个数据集。将生成的数据集顺序固定，通过依次计算当第一个数据集作为测试集，其它的数据集作为训练集；然后第二个数据集作为测试集，其它的数据集作为训练集；知道最终将第k个数据集作为测试集，其它的数据集作为训练集。这样我们就会进行k次训练，然后通过给定的评分函数，计算k次训练的分数，最终求平均值即可。
  • 网格搜索会产生所有的给定框架参数的结合，然后计算每种组合下当前框架的K折交叉验证在给定评定函数下的分数，获取所有组合在指定评定函数下的最好组合，即是我们最佳参数组合。Kfold在默认情况下是对数据进行顺序分割。
  • 当前数据的验证集、预测集的分数以及分数的平均值、标准方差。各个特征值对应的数据以及包含所有特征值数据的变量。
  • 产生的特征值可能不足以表现当前数据的关系。交叉验证多次对同一个数据进行不同的拆分来避免这个问题，由于每次产生的数据集不同所有每次通过网格搜索产生的关系也会不同，这样就可以综合评定这个值了。

模型健壮性¶

一个最优的模型不一定是一个健壮模型。有的时候模型会过于复杂或者过于简单，以致于难以泛化新增添的数据；有的时候模型采用的学习算法并不适用于特定的数据结构；有的时候样本本身可能有太多噪点或样本过少，使得模型无法准确地预测目标变量。这些情况下我们会说模型是欠拟合的。
